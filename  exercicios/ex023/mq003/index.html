Mude a orientação do seu dispositivo

Media query é = media type + media features.

media type é o tipo de midia. Os media types especificam o tipo geral de mídia ou dispositivo no qual o conteúdo será renderizado. Eles são usados para determinar em qual ambiente a media query se aplica, como uma tela, impressora, entre outros.

As media features verificam características específicas do dispositivo ou do ambiente. Elas permitem definir condições mais detalhadas, como largura da tela, altura da viewport, orientação, resolução, entre outros.
